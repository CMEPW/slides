## Marp

Install:

    npm install -g @marp-team/marp-cli

Generate HTML slides:

    marp docs/prez.md -o docs/index.html

Serve prez in live and go to http://localhost:8080/prez.md:

    marp -s docs

Generate PDF slides:

    marp --pdf docs/prez.md -o docs/prez.pdf --allow-local-files
